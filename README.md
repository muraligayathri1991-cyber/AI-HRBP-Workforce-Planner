# 🤖 AI Workforce Assistant

An AI-powered HR analytics and workforce planning application developed using Python and Streamlit. The application enables HR professionals and business leaders to explore workforce data, generate analytics, search employee information, and receive AI-driven HRBP insights through natural language queries.

---

## 📌 Project Overview

The AI Workforce Assistant is designed as a portfolio project to demonstrate practical HR analytics, workforce planning, and AI integration skills. It combines interactive dashboards with conversational AI to simplify workforce analysis and support HR decision-making.

---

## ✨ Key Features

### 👥 Employee Search
- Search employees using natural language.
- Filter by Department, Location, Designation, Gender and Experience.
- Supports multiple filters in a single query.

### 📊 Workforce Analytics
- Department-wise Headcount
- Location-wise Headcount
- Designation-wise Headcount
- Gender Distribution
- Average Salary Analysis
- Average Experience Analysis
- Department-wise and Designation-wise Diversity Reports

### 🎂 Employee Insights
- Upcoming Birthdays
- Recent Joiners

### 🤖 AI HRBP Assistant
- Answers workforce-related questions.
- Provides HR Business Partner recommendations.
- Generates practical HR insights based on workforce data.

### 📈 Data Visualization
- Interactive charts
- Executive summaries
- Workforce metrics

### 📄 Report Export
- Export reports to Excel
- Export reports to PDF
- Dynamic report filenames
- Report generation timestamp
